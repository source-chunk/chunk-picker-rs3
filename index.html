Patch Notes
2/20/2024 (6.0.0)

Major Changes & New Features

  • Chunk Sections* - Chunk Sections are the parts that make up a whole chunk, created by a river or fence separating a chunk into pieces. Now that we are able to determine not just what chunks you have access to, but what part of those chunks you have access to, we can much more accurately generate tasks and more! (This feature was a very big undertaking, and surely will have some issues. Report any you find using the Bug Report link in the top bar).

    An example of Chunk Sections in chunk 12596 on the OSRS map
  • Mobile Support - Added much better support for the Chunk Picker on mobile, allowing for nearly everything that can be done on the desktop site to be done on the mobile site too!
  • BiS Images - Added equipment images to the BiS Equipment window, to more easily check out what gear is listed there visually.
  • Map Overlays* - Added an Overlays button at the bottom-left of the map, to show different overlays overtop your chunk map.
  • Chunk Notes Markdown - Changed the Chunk Notes window to have basic support for markdown, allowing more options to customize what you want to display there.
  • Chunk Numbering - Changed the way that Selected chunk numbering works, so now it stays the same between reloads of the page. This will cut out confusion when other people look at your map also, since the numbers will be the same for you and them.
  • OSRS Friends - Added the ability to add both RS3 and OSRS Chunk Maps to your Chunk Picker Friends List.
  • Backend Logic - Overhauled the underlying logic for handling special characters in task and item names.

Other Changes & Bug Fixes

  • Added a way to select the number of doable clue steps already obtained within the Clue window.
  • Added support for pressing the Esc key to exit windows within the Chunk Picker (thanks Happery).
  • Added pre-loading for chunk images before a fancy roll, to prevent them from being partially loaded as the roll happens.
  • Added support for double-searching shops to see their chunk location.
  • Added the ability to search partial data before tasks load in the Search window.
  • Added console info output of the current version number.
  • Added a setting to enable warnings when clicking the 'Pick Chunk' button (thanks Kono Tyran).
  • Added wiki links to the Methods and All Tasks lists.
  • Added bird nest loot to the All Droptables rule.
  • Renamed 'Connections' to 'Connected Areas' in the Chunk Info sidebar.
  • Fixed Slayer-monster drops not working with double-search.
  • Fixed an issue with special characters in the Search window.
  • Fixed an issue with Every Drop tasks not including the correct title if the task name has special characters.
  • Fixed an issue where Future Tasks were calculated too often.
  • Fixed an issue with missing items from slayer monsters in All Droptables and Every Drop rules.

* Some features will require more work to be fully operational.
As always, if you experience any issues with your map or anything wrong related to this update, please contact me on Discord at whitecatblack or submit a bug report here.

Chunk Sections
Select Chunk Sections
Select the sections of the chunk you currently have access to. This helps the website be *much* more accurate at generating correct chunk tasks. What are Chunk Sections? Chunk Sections are the little pieces that make up a whole chunk, for when a chunk is not fully continous. When a fence or river cuts through a chunk and prevents you from being able to access parts of it, that's a Chunk Section. Indicating what parts of a chunk you have access to helps the Chunk Task calculation be as accurate as it can be, not giving you tasks for things in your chunk you can't access.

The website automatically figures out what parts of all your chunks you have access to, but sometimes it will get stuck and ask you to indicate what parts of a chunk you can access. It is not necessary for you to manually do this for every chunk, only if the website asks you for help or if you need to correct something it got wrong.
